* Que es un Makefile.
- make :: Es una utilidad que determina que archivos de cof=digo deben ser recomplilados y como.
- makefile :: Es un archivo que indica a =make= como recompilar mi proyecto.

Los makefiles pueden ser llamados de la siguientes maneras:
- GNUmakefile
- makefile
- Makefile

* Como se hace un makefile.
- Reglas.
- Variables.
- Patrones.
- Generacion de dependencias.
- gcc
  - Flags.
  - *.o files (archivos de codigo objeto)

* Ejemplo de Makefile para python:
El siguiente ejemplo es el entorno virtual y la instalacion de =requirements.txt= para el desarrollo de una aplicacion de =python=.
#+begin_src makefile
VENV_DIR := .venv
PYTHON := python3
PIP := $(VENV_DIR)/bin/pip

.PHONY: all venv deps clean

all: venv deps

# Create a virtual environment
venv:
	@echo "Creating virtual environment..."
	$(PYTHON) -m venv $(VENV_DIR)

# Install dependencies in the virtual environment
deps: venv
	@echo "Installing dependencies..."
	$(PIP) install --upgrade pip
	$(PIP) install -r requirements.txt

# Clean up the virtual environment
clean:
	@echo "Cleaning up virtual environment..."
	rm -rf $(VENV_DIR)
#+end_src
